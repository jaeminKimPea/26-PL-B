 ## Debuger
 - 프로그램 에러 위치 찾음
    - F9 중단점 설정/해제
    - F5 디버깅 시작
    - F10 함수단위 실행
    - F11 함수 내부진행
    - Shift + F5 디버그 중지
    - Watch 변수 값을 보여줌
    - Call stack 함수 호출 순서를 보여줌
- GDB
    - run (r) 디버깅 시작
    - break (b) 중단점 설정
    - clear 중단점 해제
    - next (n) 다음 명령 실행, 함수도 한번에 실행
    - step (s) 다음 명령 실행, 함수의 경우 함수 내부로 이동
    - print (p) 변수 값 출력
    - cont (c) 프로그램 실행 계속 (다음 중단점까지 실행 후 멈춤)
    - list (I) 프로그램 코드 보여주기

#### 명령어
- mkdir 디렉토리 생성
- ls 파일 리스트
- cd 디렉토리 변경
- pwd - 현재 디렉토리

#### 편집기 명령어
- 편집
    - i, a, o, O
- 커서 이동
    - h, j, k, l, 0, $: 왼쪽, 아래쪽, 위쪽, 오른쪽, 라인 맨 앞, 라인 맨 뒤
    - nG: n번째 라인으로 이동
    - Shift + G: 파일 제일 아래 부분으로 이동
#### 컴파일러
> cc hello.c → a.out (실행가능 파일)
> cc -c hello.c→ hello.o (목적 파일)
> c -o hello hello.c → hello (실행가능 파일)
> cc -g -o hello hello.c → hello (실행가능, 디버그 가능 파일)
